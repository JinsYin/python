# traceback 模块

## 函数

| 函数          | 描述                                                                                                        |
| ------------- | ----------------------------------------------------------------------------------------------------------- |
| format_exc()  | 返回错误堆栈字符串                                                                                          |
| print_exc()   | 打印错误堆栈字符串；还可以接受file参数直接写入到一个文件,如 `traceback.print_exc(file=open('tb.txt','w+'))` |
| print_stack() | 打印调用栈信息（不论异常与否）                                                                              |
